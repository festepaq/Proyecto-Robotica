El equipo de trabajo decidió implementar el proceso con una celda robótica compuesta por 1 robot; es necesario entonces hacer la selección del brazo robótico que realizará las tareas requeridas a lo largo del proceso. Para ello hicimos uso de la herramienta Robot Selector de ABB: Como aplicación se seleccionó Material Handling:

Robot: ABB IRB 1660

Luego se seleccionó un payload de máximo 7 Kg, suficiente para realizar los procesos necesarios en la celda robotizada:

Robot: ABB IRB 1660

Finalmente se escoge un valor para reach de hasta 1.8 m, lo cual es suficiente para realizar las tareas que componen el proceso:

Robot: ABB IRB 1660

Dados estos valores la herramienta preselecciona ocho modelos de brazos robóticos:

Robot: ABB IRB 1660

El equipo se decantó por el uso del modelo IRB 1660, puesto que los demás modelos no tienen un buen rango de alcance, además que el IRB 1660 se encuentra en el software de simulación RobotStudio.

Robot: ABB IRB 1660
